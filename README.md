# LongXia Tech Blog

Technical blog for LongXia Group's AI & automation empire.

## Features

- Static site generated by Jekyll
- GitHub Pages hosting
- Utterances comment system (GitHub Issues)
- Social sharing buttons
- Minima theme

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

## Writing Posts

Add Markdown files to `_posts/` with filename format: `YYYY-MM-DD-title.md`.

Front matter example:
```yaml
---
layout: post
title: "Your Title"
date: 2026-03-01 12:00:00 +0100
categories: technology
---
```

## Deployment

Automatically deployed to GitHub Pages when pushing to `main` branch.

Access at: https://longxiaagent.github.io/blog/
